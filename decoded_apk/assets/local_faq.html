General
How do I use it?

Alarmy offers multiple alarm modes to force you out of bed. When you add an alarm, choose your desired method from "Alarm off method". For example, if you choose "Take a picture", you must take a picture that matches a preset picture to dismiss the alarm. If the alarm mode is set to "Shake", you must shake the phone a set number of times, and so on.

Alarmy would not work for me if I turned off my phone or deleted this app when the alarm rings!

Alarmy provides "Prevent phone turn-off" and "Prevent app uninstall" options to prevent this problem. You can turn on the options in Settings.

What if a picture mode alarm rings when I can't take the same picture?
  • If you are away from your normal residence, restart your phone. Upon restarting, a new button will show on the top right corner of the alarm screen. Press this button to enter the emergency mode, which requires clicking the screen a set number of times to dismiss. Press the button again to toggle between the original alarm mode and emergency mode.
  • Another possible solution would be to turn on the "Auto-Silence" mode, which automatically mutes the phone after a set period. Go to Settings and give it a try!
My picture is not recognized correctly.
  • Picture accuracy can be set in "Settings -> Accuracy of the Picture".
  • If picture recognition continues to fail, please retry with a different picture.
  • Please note that locations with little variation in brightness are optimal for taking pictures. Also, taking pictures too close could also cause this problem!
How do I use the Barcode/QR Code Mode?
  • Please register a code on some household object, such as a toothpaste, or a book. Any barcode can be registered. Place these objects as far away from bed as possible for maximum effect.
  • Also, you can generate and print a QR code to stick to any object or place you want. You can generate QR codes here, on (https://www.the-qrcode-generator.com/).
Troubleshooting
Sometimes the alarm doesn't ring!
  • Battery-saving / memory management apps (such as 360 Security, Clean Master etc.) force stops Alarmy, causing alarms not to ring. If you are using such applications, please configure them to not force stop Alarmy :) If the alarms continue not to ring, please contact us!
  • In some devices such as Xiaomi, Huawei, and Sony, periodic cleaning performed by the system will kill the Alarmy. Please follow the instructions on the "Caution" page. (Alarmy → Upper left menu → Caution).
I keep getting a <Screen overlay detected> message! (for Android Marshmallow Users)

Screen filter apps such as Bluelight Filter may cause this problem. These apps provide an overlay over the phone screen, hence displaying this message. When you see this message, please turn off your filter momentarily. (Please understand that it is Android policy with overlays and is thus unavoidable.)

Can't uninstall this app?

It is not possible to delete Alarmy if "Prevent app uninstall" option is turned on. Please check it in Setttings!
